### 📝 HTML 구조 특징:

**📱 반응형 레이아웃**
- Grid와 Flexbox를 활용한 모던 레이아웃
- 모바일 친화적인 overflow 처리
- 다양한 화면 크기에 대응

**🎯 주요 섹션들**
1. **헤더**: 브랜딩 및 소개
2. **메뉴 관리**: 상품 추가/삭제 기능
3. **참가자 관리**: 인원 추가/삭제 기능  
4. **소비 매핑**: 누가 무엇을 소비했는지 입력

6. **결과 표시**: 동적으로 생성되는 계산 결과

- 직관적인 아이콘과 이모지 사용

- 입력 검증을 위한 적절한 input 속성들
- 실시간 자동 계산 옵션 제공
